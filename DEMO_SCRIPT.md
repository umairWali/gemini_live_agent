## Technical Details for Judges

### API Endpoints Demonstrated
```
POST /api/storyteller    # Interleaved multimedia
POST /api/navigator      # Visual UI automation
POST /api/swarm          # Multi-agent execution
WS  /ws/                 # Live audio streaming
```

### Gemini Models Used
- `gemini-2.5-flash-native-audio-latest` - Live voice
- `gemini-2.0-flash-exp-image-generation` - Image generation
- `gemini-2.0-flash` - Vision analysis

### Google Cloud Services
- Cloud Run (serverless hosting)
- Cloud Build (container building)
- Container Registry (image storage)

### Key Innovation Points
1. **True Interleaved Output** - Not sequential, but mixed media in real-time
2. **Native Audio Streaming** - 16kHz PCM, not STT/TTS pipeline
3. **Visual + Action** - UI understanding with automated interaction
4. **Parallel Agent Execution** - 6 agents working simultaneously
5. **Self-Healing** - Automatic code repair without human intervention

## Recording Tips

1. **Use OBS or Screen Studio** for smooth recording
2. **Test voice commands beforehand** - make sure mic works
3. **Have backup screenshots** in case live generation is slow
4. **Show real-time aspect** - don't cut during generation
5. **Keep it under 4 minutes** - practice timing
